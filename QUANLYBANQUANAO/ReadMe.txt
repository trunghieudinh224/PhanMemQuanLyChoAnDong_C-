Cấu hình tối thiểu để chạy setup: 1920 x 1080 (Cấu hình thấp hơn sẽ bị mất đi 1 phần giao diện).
---------------
1. Mở file CaiDatPhanMem\Quan Ly Ban Quan Ao Cho An Dong\Debug\setup.exe.
2. Chạy setup.
3. Thêm 2 folder "Toa" và "PhieuNhap" vào folder khởi tạo của phần mềm sau khi cài đặt.
4. Chạy phần mềm.
-------
Database sẽ nằm chung trong thư mục của phần mềm.
